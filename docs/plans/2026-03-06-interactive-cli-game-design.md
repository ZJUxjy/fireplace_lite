# 交互式炉石传说 CLI 游戏设计

## 概述
创建 `play.py` 单文件脚本，实现玩家 vs AI 的交互式炉石传说游戏。

## 需求
- 游戏模式: 玩家 vs AI
- 卡组: 随机生成
- 界面: 简洁文字模式
- 换牌: 不需要

## 游戏流程
1. 初始化 - 随机选择职业和卡组
2. 开始游戏 - 跳过换牌
3. 回合循环 - 玩家操作 → AI 操作
4. 游戏结束 - 显示结果

## 命令系统
- `play <编号>` - 打牌
- `attack <己方> <敌方>` - 攻击
- `hero [目标]` - 英雄技能
- `end` - 结束回合
- `status` - 显示场况
- `help` - 帮助
- `quit` - 退出

## AI 逻辑
简单随机策略：随机打牌、随机攻击、有概率用技能。

## 文件
- `play.py` - 主程序（约 200-300 行）
