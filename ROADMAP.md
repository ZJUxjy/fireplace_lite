# Fireplace WebUI 开发路线图

## 当前版本: v0.2.0-alpha (Core Gameplay Milestone)

---

## v0.3.0 目标: 完整职业机制 + 英雄牌系统

### Phase 1: 核心缺失机制 (P0)
**目标: 完善基础游戏体验**

#### 1.1 冲锋与突袭 (Charge & Rush)
- [ ] 冲锋 - 登场回合可攻击英雄
- [ ] 突袭 - 登场回合可攻击随从
- [ ] 视觉指示器（绿光环绕）
- [ ] 回合结束重置攻击次数

#### 1.2 沉默 (Silence)
- [ ] 移除所有卡牌文本
- [ ] 清除 buff/debuff
- [ ] 保留基础攻击力/生命值
- [ ] 视觉反馈（沉默图标）

#### 1.3 免疫 (Immune)
- [ ] 无法成为目标
- [ ] 不受到任何伤害
- [ ] 视觉指示器（金色护盾）

#### 1.4 游戏边界规则
- [ ] 手牌上限 10 张（多余烧毁）
- [ ] 牌库抽空 - 疲劳伤害（1,2,3...）
- [ ] 场上随从上限 7 个
- [ ] 回合时间限制（烧绳机制）

**目标: 每个职业有独特玩法**

#### 3.1 德鲁伊 (Druid)

#### 3.2 猎人 (Hunter)

- [ ] 亡语猎体系

#### 3.3 法师 (Mage)
- [ ] 发现法术 (Spell Discovery)
- [ ] 冻结控制体系

#### 3.4 圣骑士 (Paladin)

#### 3.5 牧师 (Priest)

#### 3.8 术士 (Warlock)
